# Descify：AI搭載のShopify商品説明ジェネレーター

**Descify**は、Eコマース事業者向けのコピーライティングプロセスを自動化・最適化するために設計された包括的なShopifyアプリケーションです。高度なAIを活用することで、Descifyは店舗オーナーが高品質でSEOに最適化され、コンバージョンに焦点を当てた商品説明を一括生成し、手作業の時間を大幅に削減します。

## 🚀 主要機能の概要

### 1. ホームページ：一目でわかるインサイト
**ホームページ**は、使用状況と節約時間をリアルタイムで把握できるコマンドセンターです。

<div align="center">

![Home Page Dashboard](images/1.%20Home%20Page.png)
</div>

#### 番号とラベルの説明
1. **Generated Words（生成された単語数）**  
   Descifyによって生成された総単語数
2. **Saved Your Time（節約された時間）**  
   Descifyの支援により節約された時間
3. **Usable Credits（利用可能なクレジット）**  
   残りの利用可能なクレジット数
4. **Your Plan（ご利用プラン）**  
   現在のメンバーシップタイプ
5. **Recent History（最近の履歴）**  
   最新の5つの生成タスクを素早く確認し、即座の進捗を追跡できます。
   ここをクリックして生成履歴に移動

---

### 2. 商品選択
**商品選択**ビューで、カタログを簡単に管理し、生成設定を構成できます。

<div align="center">

![Product List Selection](images/2.%20Product%20List.png)
</div>

#### 番号とラベルの説明

1. **Search Box（検索ボックス）**  
   商品キーワードを入力して、説明を最適化する商品を検索します。
2. **Click to Search（検索をクリック）**  
   ここをクリックして検索を実行します。
3. **Template Selection Preview（テンプレート選択プレビュー）**  
   利用可能な説明テンプレートを閲覧・プレビューします。
    * **9つのユニークなテンプレート：** *プロフェッショナル権威*、*シナリオストーリー*、*機能・利点・ベネフィット*、*ラグジュアリープレステージ*など、ブランドに最適なトーンを選択できます。
    * **多言語サポート：** 20の異なる言語をサポートし、ストアのローカライズを実現します。

<div align="center">

![Template Selection](images/3.%20Product%20List-Templates.png)

</div>


4. **Selection Mode（選択モード）**  
   単一選択またはすべて選択オプションから選択します。
5. **Page Size Configuration（ページサイズ設定）**  
   1ページあたり10/25/50アイテムの動的切り替えをサポートします。
6. **Page Navigation（ページナビゲーション）**  
   ここをクリックして異なるページ間を移動します。
7. **Initiate Generation（生成を開始）**  
   商品、テンプレート、希望言語を選択した後、ここをクリックして説明生成プロセスを開始します。

---

### 3. 履歴
**履歴**は、コンテンツをレビュー、改善、公開する場所です。強力なフィルタリングと編集機能を提供します。

<div align="center">

![History Management](images/4.%20History.png)

</div>

#### 番号とラベルの説明
1. **Search Box（検索ボックス）**  
   商品名キーワードでフィルタリングします。

2. **Filter by Generation Status（生成ステータスでフィルタ）**  
   現在の生成フェーズに基づいて結果を絞り込みます。

<div align="center">

![Filter by Generation Status](images/5.%20History%20by%20Status.png)

</div>

3. **Filter by Template Type（テンプレートタイプでフィルタ）**  
   使用されたテンプレートカテゴリに応じて選択を絞り込みます。

<div align="center">

![Filter by Template Type](images/6.%20History%20by%20Templates.png)

</div>


4. **Selection Options（選択オプション）**  
   単一選択またはすべて選択チェックボックス。

5. **Regenerate（再生成）**  
   選択した生成レコードに満足できない場合、「再生成」ボタンをクリックして商品選択ページに素早く戻り、テンプレートと言語を再選択して商品説明を新たに生成できます。

6. **Sync to Product（商品に同期）**  
   生成された説明を対応する商品に同期します。

7. **Product Thumbnail and Name（商品サムネイルと名前）**  
   現在の生成レコードに関連付けられた商品のサムネイルと名前を表示します。

8. **Edit Mode（編集モード）**  
   ここをクリックして編集モードに入り、Descifyが提供する軽量エディタで生成されたコンテンツを修正・保存できます。
   
<div align="center">

![Edit Mode](images/7.%20History%20by%20Editor.png)

</div>

9. **Page Size Adjustment（ページサイズ調整）**  
   オプションを使用して1ページあたりのアイテム数を調整：10、25、または50。

10. **Product Name（商品名）**  
    商品の名前。

11. **Generation Language（生成言語）**  
    この生成サイクルで選択された言語。

12. **Generation Status（生成ステータス）**  
    以下のステータスは生成プロセスの現在のフェーズを示します：
    - **Pending（保留中）**：処理待ち。
    - **In Processing（処理中）**：生成が進行中。
    - **Waiting for Apply（適用待ち）**：生成完了、商品への同期待ち。
    - **Applied（適用済み）**：商品への同期が成功。
    - **Failed（失敗）**：エラーが発生し、生成が失敗しました。
