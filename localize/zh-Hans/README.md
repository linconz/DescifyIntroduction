# Descify：AI 驱动的 Shopify 产品描述生成器

**Descify** 是一款全面的 Shopify 应用程序，旨在为电子商务商家自动化并优化文案编写流程。利用先进的 AI 技术，Descify 帮助店主不仅能批量生成高质量、SEO 友好且专注于转化率的产品描述，还能节省数小时的手工工作时间。

## 🚀 主要功能概览

### 1. 首页：洞察尽在掌握
**首页**是您的控制中心，提供有关您的使用情况和节省成本的实时概览。

<div align="center">

![Home Page Dashboard](images/1.%20Home%20Page.png)
</div>

#### 序号与标签说明
1. **Generated Words (生成的字数)**  
   Descify 生成的字数总和
2. **Saved Your Time (为您节省的时间)**  
   使用 Descify 节省的时间
3. **Usable Credits (可用点数)**  
   剩余的可用点数
4. **Your Plan (您的套餐)**  
   您当前的会员类型
5. **Recent History (近期记录)**  
   快速查看您最近的 5 个生成任务，以便跟踪即时进度。
   点击此处跳转至生成历史记录

---

### 2. 产品选择
在**产品选择**视图中，轻松管理您的目录并配置生成设置。

<div align="center">

![Product List Selection](images/2.%20Product%20List.png)
</div>

#### 序号与标签说明

1. **Search Box (搜索框)**  
   通过输入产品关键词来定位需要优化描述的产品。
2. **Click to Search (点击搜索)**  
   点击此处执行搜索。
3. **Template Selection Preview (模板选择预览)**  
   浏览并预览可用的描述模板。
    * **9 种独特模板：** 选择适合您品牌的完美基调，例如*专业权威*、*场景故事*、*功能优势利益* 或 *奢华尊贵*。
     * **多语言支持：** 支持 20 种不同的语言，助您实现店铺本地化。

<div align="center">

![Template Selection](images/3.%20Product%20List-Templates.png)

</div>


4. **Selection Mode (选择模式)**  
   在单选或全选选项之间进行选择。
5. **Page Size Configuration (每页显示数量配置)**  
   支持在每页 10/25/50 个项目之间动态切换。
6. **Page Navigation (页面导航)**  
   点击此处在不同页面之间导航。
7. **Initiate Generation (开始生成)**  
   选择产品、模板和首选语言后，点击此处开始产品描述生成流程。

---

### 3. 历史记录
**历史记录**是您查看、优化和发布内容的地方。它提供强大的筛选和编辑功能。

<div align="center">

![History Management](images/4.%20History.png)

</div>

#### 序号与标签说明
1. **Search Box (搜索框)**  
   按产品名称关键词筛选。

2. **Filter by Generation Status (按生成状态筛选)**  
   根据当前的生成阶段缩小结果范围。

<div align="center">

![Filter by Generation Status](images/5.%20History%20by%20Status.png)

</div>

3. **Filter by Template Type (按模板类型筛选)**  
   根据使用的模板类别筛选。

<div align="center">

![Filter by Template Type](images/6.%20History%20by%20Templates.png)

</div>


4. **Selection Options (选择选项)**  
   单选或全选复选框。

5. **Regenerate (重新生成)**  
   如果不满意选定的生成记录，点击“重新生成”按钮可快速返回产品选择页面，您可以在那里重新选择模板和语言，重新生成产品描述。

6. **Sync to Product (同步至产品)**  
   将生成的描述同步到相应的产品。

7. **Product Thumbnail and Name (产品缩略图和名称)**  
   显示与当前生成记录关联的产品缩略图和名称。

8. **Edit Mode (编辑模式)**  
   点击此处进入编辑模式，您可以在此通过 Descify 提供的精简轻量级编辑器修改并保存生成的内容。
   
<div align="center">

![Edit Mode](images/7.%20History%20by%20Editor.png)

</div>

9. **Page Size Adjustment (每页显示调整)**  
   使用选项调整每页显示的项目数量：10、25 或 50。

10. **Product Name (产品名称)**  
    您的产品名称。

11. **Generation Language (生成语言)**  
    本次生成循环中所选的语言。

12. **Generation Status (生成状态)**  
    以下状态指示生成过程的当前阶段：
    - **Pending (待处理)**：等待处理。
    - **In Processing (处理中)**：生成正在进行中。
    - **Waiting for Apply (待应用)**：生成完成，等待您同步到产品。
    - **Applied (已应用)**：成功同步到产品。
    - **Failed (失败)**：发生错误，生成未成功。
