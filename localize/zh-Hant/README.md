# Descify：AI 驅動的 Shopify 產品描述生成器

**Descify** 是一款全面的 Shopify 應用程式，旨在為電子商務商家自動化並優化文案編寫流程。利用先進的 AI 技術，Descify 幫助店主不僅能批量生成高品質、SEO 友好且專注於轉化率的產品描述，還能節省數小時的手工工作時間。

## 🚀 主要功能概覽

### 1. 首頁：洞察盡在掌握
**首頁**是您的控制中心，提供有關您的使用情況和節省成本的即時概覽。

<div align="center">

![Home Page Dashboard](images/1.%20Home%20Page.png)
</div>

#### 序號與標籤說明
1. **Generated Words (生成的字數)**  
   Descify 生成的字數總和
2. **Saved Your Time (為您節省的時間)**  
   使用 Descify 節省的時間
3. **Usable Credits (可用點數)**  
   剩餘的可用點數
4. **Your Plan (您的方案)**  
   您目前的會員類型
5. **Recent History (近期記錄)**  
   快速查看您最近的 5 個生成任務，以便追蹤即時進度。
   點擊此處跳轉至生成歷史記錄

---

### 2. 產品選擇
在**產品選擇**視圖中，輕鬆管理您的目錄並配置生成設定。

<div align="center">

![Product List Selection](images/2.%20Product%20List.png)
</div>

#### 序號與標籤說明

1. **Search Box (搜尋框)**  
   透過輸入產品關鍵字來定位需要優化描述的產品。
2. **Click to Search (點擊搜尋)**  
   點擊此處執行搜尋。
3. **Template Selection Preview (範本選擇預覽)**  
   瀏覽並預覽可用的描述範本。
    * **9 種獨特範本：** 選擇適合您品牌的完美基調，例如*專業權威*、*場景故事*、*功能優勢利益* 或 *奢華尊貴*。
    * **多語言支援：** 支援 20 種不同的語言，助您實現店舖在地化。

<div align="center">

![Template Selection](images/3.%20Product%20List-Templates.png)

</div>


4. **Selection Mode (選擇模式)**  
   在單選或全選選項之間進行選擇。
5. **Page Size Configuration (每頁顯示數量配置)**  
   支援在每頁 10/25/50 個項目之間動態切換。
6. **Page Navigation (頁面導覽)**  
   點擊此處在不同頁面之間導覽。
7. **Initiate Generation (開始生成)**  
   選擇產品、範本和首選語言後，點擊此處開始產品描述生成流程。

---

### 3. 歷史記錄
**歷史記錄**是您檢閱、優化和發佈內容的地方。它提供強大的篩選和編輯功能。

<div align="center">

![History Management](images/4.%20History.png)

</div>

#### 序號與標籤說明
1. **Search Box (搜尋框)**  
   按產品名稱關鍵字篩選。

2. **Filter by Generation Status (按生成狀態篩選)**  
   根據目前的生成階段縮小結果範圍。

<div align="center">

![Filter by Generation Status](images/5.%20History%20by%20Status.png)

</div>

3. **Filter by Template Type (按範本類型篩選)**  
   根據使用的範本類別篩選。

<div align="center">

![Filter by Template Type](images/6.%20History%20by%20Templates.png)

</div>


4. **Selection Options (選擇選項)**  
   單選或全選核取方塊。

5. **Regenerate (重新生成)**  
   如果不滿意選定的生成記錄，點擊「重新生成」按鈕可快速返回產品選擇頁面，您可以在那裡重新選擇範本和語言，重新生成產品描述。

6. **Sync to Product (同步至產品)**  
   將生成的描述同步到相應的產品。

7. **Product Thumbnail and Name (產品縮圖和名稱)**  
   顯示與目前生成記錄關聯的產品縮圖和名稱。

8. **Edit Mode (編輯模式)**  
   點擊此處進入編輯模式，您可以在此透過 Descify 提供的精簡輕量級編輯器修改並儲存生成的內容。
   
<div align="center">

![Edit Mode](images/7.%20History%20by%20Editor.png)

</div>

9. **Page Size Adjustment (每頁顯示調整)**  
   使用選項調整每頁顯示的項目數量：10、25 或 50。

10. **Product Name (產品名稱)**  
    您的產品名稱。

11. **Generation Language (生成語言)**  
    本次生成循環中所選的語言。

12. **Generation Status (生成狀態)**  
    以下狀態指示生成過程的目前階段：
    - **Pending (待處理)**：等待處理。
    - **In Processing (處理中)**：生成正在進行中。
    - **Waiting for Apply (待應用)**：生成完成，等待您同步到產品。
    - **Applied (已應用)**：成功同步到產品。
    - **Failed (失敗)**：發生錯誤，生成未成功。
