# Descify: AI 기반 Shopify 제품 설명 생성기

**Descify**는 전자상거래 판매자를 위한 카피라이팅 프로세스를 자동화하고 최적화하도록 설계된 포괄적인 Shopify 애플리케이션입니다. 고급 AI를 활용하여 Descify는 매장 소유자가 고품질의 SEO 친화적이며 전환에 중점을 둔 제품 설명을 대량으로 생성하여 수작업 시간을 절약할 수 있도록 돕습니다.

## 🚀 주요 기능 개요

### 1. 홈페이지: 한눈에 보는 인사이트
**홈페이지**는 사용량과 절약 시간에 대한 실시간 개요를 제공하는 명령 센터입니다.

<div align="center">


![Home Page Dashboard](images/1.%20Home%20Page.png)

</div>


#### 일련번호 및 레이블 설명
1. **Generated Words (생성된 단어 수)**  
   Descify가 생성한 총 단어 수
2. **Saved Your Time (절약된 시간)**  
   Descify의 지원으로 절약된 시간
3. **Usable Credits (사용 가능한 크레딧)**  
   남은 사용 가능한 크레딧 수
4. **Your Plan (요금제)**  
   현재 멤버십 유형
5. **Recent History (최근 기록)**  
   최신 5개의 생성 작업을 빠르게 확인하여 즉각적인 진행 상황을 추적합니다.
   여기를 클릭하여 생성 기록으로 이동

---

### 2. 제품 선택
**제품 선택** 보기에서 카탈로그를 쉽게 관리하고 생성 설정을 구성할 수 있습니다.

<div align="center">


![Product List Selection](images/2.%20Product%20List.png)

</div>


#### 일련번호 및 레이블 설명

1. **Search Box (검색창)**  
   제품 키워드를 입력하여 설명 최적화가 필요한 제품을 찾습니다.
2. **Click to Search (검색 클릭)**  
   여기를 클릭하여 검색을 실행합니다.
3. **Template Selection Preview (템플릿 선택 미리보기)**  
   사용 가능한 설명 템플릿을 탐색하고 미리 봅니다.
    * **9가지 고유 템플릿:** *전문 권위*, *시나리오 스토리*, *기능 이점 혜택*, *럭셔리 프레스티지* 등 브랜드에 완벽한 톤을 선택하세요.
    * **다국어 지원:** 20개의 다른 언어를 지원하여 매장을 현지화합니다.

<div align="center">

![Template Selection](images/3.%20Product%20List-Templates.png)


</div>



4. **Selection Mode (선택 모드)**  
   단일 선택 또는 전체 선택 옵션 중에서 선택합니다.
5. **Page Size Configuration (페이지 크기 구성)**  
   페이지당 10/25/50개 항목 간의 동적 전환을 지원합니다.
6. **Page Navigation (페이지 탐색)**  
   여기를 클릭하여 다른 페이지 간을 이동합니다.
7. **Initiate Generation (생성 시작)**  
   제품, 템플릿 및 선호 언어를 선택한 후 여기를 클릭하여 설명 생성 프로세스를 시작합니다.
8. **Description Length (설명 길이)**
   
   Descify를 사용하면 생성된 제품 설명의 길이를 사용자 정의할 수 있으며, 선택할 수 있는 세 가지 길이 옵션을 제공합니다.
   
   **Short (짧음)** 약 200단어
   
   **Medium (중간)** 약 400단어
   
   **Long (김)** 약 600단어

### 3. 기록
**기록**은 콘텐츠를 검토하고 개선하며 게시하는 곳입니다. 강력한 필터링 및 편집 기능을 제공합니다.

<div align="center">

![History Management](images/4.%20History.png)


</div>


#### 일련번호 및 레이블 설명
1. **Search Box (검색창)**  
   제품 이름 키워드로 필터링합니다.

2. **Filter by Generation Status (생성 상태별 필터)**  
   현재 생성 단계를 기준으로 결과를 좁힙니다.

<div align="center">

![Filter by Generation Status](images/5.%20History%20by%20Status.png)


</div>


3. **Filter by Template Type (템플릿 유형별 필터)**  
   사용된 템플릿 카테고리에 따라 선택을 세분화합니다.

<div align="center">

![Filter by Template Type](images/6.%20History%20by%20Templates.png)


</div>



4. **Selection Options (선택 옵션)**  
   단일 선택 또는 전체 선택 체크박스.

5. **Regenerate (재생성)**  
   선택한 생성 기록에 만족하지 않으면 "재생성" 버튼을 클릭하여 제품 선택 페이지로 빠르게 돌아가 템플릿과 언어를 다시 선택하여 제품 설명을 새로 생성할 수 있습니다.

6. **Sync to Product (제품에 동기화)**  
   생성된 설명을 해당 제품에 동기화합니다.

7. **Product Thumbnail and Name (제품 썸네일 및 이름)**  
   현재 생성 기록과 연결된 제품의 썸네일과 이름을 표시합니다.

8. **Edit Mode (편집 모드)**  
   여기를 클릭하여 편집 모드로 들어가 Descify가 제공하는 간소화된 경량 편집기로 생성된 콘텐츠를 수정하고 저장할 수 있습니다.
   
<div align="center">

![Edit Mode](images/7.%20History%20by%20Editor.png)


</div>


9. **Page Size Adjustment (페이지 크기 조정)**  
   옵션을 사용하여 페이지당 항목 수를 조정: 10, 25 또는 50.

10. **Product Name (제품 이름)**  
    제품의 이름.

11. **Generation Language (생성 언어)**  
    이 생성 주기에서 선택한 언어.

12. **Generation Status (생성 상태)**  
    다음 상태는 생성 프로세스의 현재 단계를 나타냅니다:
    - **Pending (대기 중)**: 처리 대기 중.
    - **In Processing (처리 중)**: 생성 진행 중.
    - **Waiting for Apply (적용 대기)**: 생성 완료, 제품 동기화 대기 중.
    - **Applied (적용됨)**: 제품에 성공적으로 동기화됨.
    - **Failed (실패)**: 오류가 발생하여 생성이 실패했습니다.

## 💡 판매자에게 도움이 되는 방법
* **시간 절약:** 수백 개의 SKU에 대한 설명을 작성하는 반복 작업을 자동화합니다.
* **SEO 향상:** 고유하고 키워드가 풍부한 콘텐츠를 생성하여 제품 순위를 높입니다.
* **글로벌 진출:** 제품 콘텐츠를 즉시 번역하고 현지화하여 국제 시장에 진출합니다.
